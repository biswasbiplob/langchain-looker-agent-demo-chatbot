Integration Guide

Embed the Chatbot

Add this code snippet to any webpage to embed the chatbot widget:

<!-- CSS -->
<link rel="stylesheet" href="{{ request.url_root }}static/widget.css">

<!-- JavaScript -->
<script src="{{ request.url_root }}static/widget.js"></script>
<script>
  // Initialize the widget
  const chatWidget = new LookerChatWidget({
    apiBaseUrl: '{{ request.url_root.rstrip('/') }}'
  });
</script>
Environment Configuration

Configure these environment variables for Looker integration:

LOOKER_BASE_URL=https://your-looker-instance.looker.com
LOOKER_CLIENT_ID=your_client_id
LOOKER_CLIENT_SECRET=your_client_secret
OPENAI_API_KEY=your_openai_api_key
SESSION_SECRET=your_session_secret
